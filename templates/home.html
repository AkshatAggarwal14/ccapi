Unofficial Codechef API

Methods

Problemset

Returns list of problems

Example: https://apicodechef.herokuapp.com/problemset

Rating

Returns list of objects related to user rating

Parameter Description
handle (Required) User's Handle (Username). For example:/users/gennady.korotkevich

Response Time: < 1s

Example: https://apicodechef.herokuapp.com/rating?handle=gennady.korotkevich
